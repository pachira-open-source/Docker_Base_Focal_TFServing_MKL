## Includes

- Ubuntu 20.04 (Focal)
- Set language as en_US.UTF-8
- Set timezone as Asia/Shanghai
- Python 3.8(Stock)
- GCC/G++ 9 (Stock)
- Intel Optimized Tensorflow Serving (MKL) 2.2.0
